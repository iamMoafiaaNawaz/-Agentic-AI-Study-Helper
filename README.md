# -Agentic-AI-Study-Helper
"Advanced AI agent using structured tool calling and  LLM function capabilities for educational assistance.  Built with LiteLLM and native function calling."
## Assignments

### Assignment 1: Basic Agentic AI Application
- File: `study_helper_agent.ipynb`
- Features: Quiz generation, text summarization, concept explanation
- Tools: Manual JSON parsing with LiteLLM

### Assignment 2: Enhanced with Structured Tools
- File: `assignment2_enhanced_study_helper.ipynb`
- Features: Native function calling, JSON Schema, multi-parameter tools
- Improvements: Better validation, error handling

## Setup
1. Install: `pip install litellm`
2. Add API key (Groq or OpenAI)
3. Run notebooks
